How to set up cpp environment in VSCode (Windows 10)

Install MinGW

See official website and follow the instructions step by step (https://code.visualstudio.com/docs/cpp/config-mingw). Note that we need to add the folder path C:\msys64\ucrt64\bin (for example) to both User Variables and Environment Variables. Someone said the order of the path also matters, so put this path in front of others if possible.

Install CMake

Use CMake as the build tool. Download .msi from the official website. Remember to select “Add CMake to system PATH for all users” while installation so that we can use CMake in the command line.
